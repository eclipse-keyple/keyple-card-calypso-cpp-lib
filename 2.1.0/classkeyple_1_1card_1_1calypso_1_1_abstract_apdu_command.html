Detailed Description

(package-private)
Generic APDU command.

It provides the generic getters to retrieve:

  • the card command reference,
  • the name of the command,
  • the built org.calypsonet.terminal.card.spi.ApduRequestSpi,
  • the parsed org.calypsonet.terminal.card.ApduResponseApi.
